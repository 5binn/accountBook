## 가계부 시스템

### 서비스 설명

- 가계부의 내역을 등록하여 월/일/항목 별로 출력하게 함.<br>
- 회원기능을 도입하여 타 회원과 가계부를 공유할 수 있음.<br>
- 시스템 시작 시 관리자를 통해 관리되는 명언이 랜덤으로 출력됨.<br>
- 회원/가계부/내역의 CRUD<br>
<br/>
